AngularJS – the ‘A’ in MEAN

AngularJS Features


  • AngularJS is a client side MV* framework maintained by Google.
  • SPA framework capable of building an entire full-scale business type apps.  
  • View and data model are always in sync — there is no need for manual DOM manipulation.  
  • Weighs just 60kb and compatible with all modern browsers.
  • Counterparts: Backbone.js, Ember.js, Knockout.js, etc

Why AngularJS ?


  • Enhances HTML by attaching directives to your pages with new attributes or tags.
  • Provides data-binding with expressions in order to define very powerful templates directly in your HTML.
  • Encapsulates the behavior of your application in controllers which are instantiated thanks to dependency injection.
  • Helps structure and test Javascript code very easily.
  • Utility code can easily be factorized into Services/Factories that can be injected in your controllers.

Directives


  • "ng-" represents an Angular directive.  
    A directive is a marker on HTML tag that tells Angular to run or reference some javascript code.

  • Extends HTML very easily by simply adding attributes and elements.

  • Built-in directives like  ng-app, ng-model, ng-repeat, etc.

  • Support available for custom directives.

Filters

  • Provides way to apply logic to model for filtering data.  

  • Built-in filters like filter, currency, orderBy, etc. 

  • Support  available for custom filters.

Views and Controllers

  • Controllers are where we define our app's behavior by defining function and values.

  • View as we all know is our HTML tags with angular directives.

  • The glue between the View and the Controller is something called the Scope.
                         

$SCOPE

  • $scope is an object that refers to the application model.

  • C ontrollers are used to populate the scope with all the necessary data for the view.

  • Loosely Coupled: The view doesn't have to know about the controller and  vice-versa.

  • Debugging for scope objects is possible using chrome extensions AngularJS Batarang.
